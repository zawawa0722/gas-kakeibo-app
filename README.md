# gas-kakeibo-app

家計簿アプリ「ワンバンク」から書き出したCSVファイルを読み込み、Google Apps Script (GAS) を使って自動的に収支を分析・集計してGoogleスプレッドシートにまとめるシステムです。

---

## 開発の背景（作った理由）

普段利用している家計簿アプリ「ワンバンク」の画面だけでは、自分が本当に知りたい情報や詳細な分析結果をパッと一目で把握することが難しいと感じていました。

そこで、ワンバンクが持つ正確な支出データを活かしつつ、自分が一番欲しい形で情報を整理・カスタマイズして「見える化」するために、このシステムを開発しました。手動での面倒な再入力をすることなく、自動で理想の家計管理を実現します。

---

## 主な特徴

- CSVデータの自動取り込み: 「ワンバンク」アプリで出力したCSVデータをそのまま読み込み可能
- 全自動データ分析: 日時、金額、カテゴリ、支払い手段をGASが判別し、自動で集計・仕分け
- スプレッドシートで一元管理: 集計結果を使い慣れたGoogleスプレッドシートへ美しく出力

---

## 処理フロー（データの流れ）

> **1. CSVデータの用意**
>
> - 「ワンバンク」アプリから支出データ（CSVファイル）を出力します。
>
> ▼
>
> **2. GASによる読み込み・解析**
>
> - Google Apps Script（GAS）が、CSV内の情報（日時、金額、カテゴリ、支払い手段など）を1行ずつ正確に読み取ります。
>
> ▼
>
> **3. 自動集計と仕分け**
>
> - 読み取ったデータをもとに、日々の支出やカテゴリごとの合計などをシステムが自動で計算・分析します。
>
> ▼
>
> **4. 分析結果の出力**
>
> - 整理された分析結果が、指定したGoogleスプレッドシートへリアルタイムに自動で落とし込まれます。
